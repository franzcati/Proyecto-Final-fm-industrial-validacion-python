# Sistema de validación automatizado - FM Industrial

Proyecto del curso **Fundamentos de Programación I**.

## Objetivo

Procesar pedidos de fajas industriales desde un archivo CSV, validar los datos,
calcular subtotal, IGV y total, y generar un reporte para facturación y despacho.

## Archivos

- `sistema_validacion_fm.py`: programa principal.
- `pedidos_fm_industrial.csv`: datos de prueba.
- `facturas_validadas.csv`: archivo generado al ejecutar la opción 1.
- `CONTRIBUTING.md`: reglas para el trabajo colaborativo.
- `Informe_Final_FM_Industrial.docx`: informe académico del proyecto.

## Ejecución

```bash
python sistema_validacion_fm.py
```

En Windows también puede usarse:

```bash
py sistema_validacion_fm.py
```

Seleccione la opción `1` para procesar el CSV y generar el archivo de salida.

## Requisitos

- Python 3.10 o superior recomendado.
- No requiere instalar librerías externas.
